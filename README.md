## 💎 Overview

**INCENSE** is a luxury perfume store application designed to transform the elegance of oriental fragrances into a refined digital shopping experience.

The application focuses on:

- Premium User Experience (UX)
- Clean Architecture principles
- Modular widget structure
- Smooth animations
- Fully adaptive UI

This is not just a store — it is a digital luxury atmosphere.

---

## 🚀 Project Highlights

### 🌊 1. Waterfall Animation Engine

Custom `ScrollReveal` widget enables:

- Staggered animations  
- Fade & Slide transitions  
- Smooth section appearance while scrolling  

This creates a storytelling flow that guides user attention naturally.

---

### 🧩 2. Modular Component System

Each section is completely independent:

- Hero Section  
- About Section  
- Categories Section  
- Shipping Section  
- Testimonials Section  

**Benefits:**

- Easy reordering  
- Reusable components  
- Clean separation of concerns  
- Scalable structure  

---

### 📱 3. Adaptive UI

Using `flutter_screenutil`, the UI dynamically adapts to:

- Small & large Android screens  
- iPhones of all sizes  
- Different pixel densities  

Maintaining:

- Balanced spacing  
- Consistent typography  
- Golden accent precision  

---

### 📄 4. Integrated Policies Framework

Professional store policies implemented as responsive Flutter screens:

- Privacy Policy  
- Shipping Policy  
- Return & Exchange Policy  

Integrated seamlessly within the Footer to increase trust and credibility.
